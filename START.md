# 🚀 БЫСТРЫЙ СТАРТ

## 1️⃣ Открыть сайт (прямо сейчас!)

Просто откройте файл `index.html` в браузере:
- **Windows**: Двойной щелчок на `index.html`
- **macOS/Linux**: Правый клик → "Open with" → Браузер

**Готово! Сайт работает сразу с красивыми SVG плейсхолдерами.**

---

## 2️⃣ Создать красивые изображения анимаций (опционально)

Если хотите заменить плейсхолдеры на реальные красиво нарисованные картинки, запустите:

```bash
python setup.py
```

Скрипт автоматически:
- ✅ Устанавливает нужные библиотеки (если её нет)
- ✅ Генерирует PNG изображения для всех анимаций
- ✅ Обновляет галерею

---

## 3️⃣ Альтернативные способы генерации

Если `setup.py` не сработает, попробуйте:

### ✨ Вариант A: С matplotlib (рекомендуется, качественнее)
```bash
pip install matplotlib numpy
python generate_images_matplotlib.py
```

### 🎨 Вариант B: С PIL (быстрее)
```bash
pip install pillow
python create_placeholders.py
```

### 🐢 Вариант C: С оригинальным Turtle (собирает скриншоты)
```bash
pip install pillow
python generate_images.py
```

## 🐢 Как запустить анимацию

1. Откройте модальное окно кода (кнопка "👁️ Посмотреть код")
2. Скопируйте код (кнопка "📋 Копировать код")
3. Создайте файл `animation.py` и вставьте код
4. Запустите: `python animation.py`
5. Окно turtle должно открыться и нарисовать анимацию

## ❓ FAQ

**Q: Сайт не открывается?**
A: Убедитесь, что `index.html` находится в той же папке, что и `style.css` и `script.js`

**Q: Картинки остались плейсхолдерами?**
A: Запустите `python setup.py`. Библиотеки не установлены автоматически.

**Q: Как добавить свою анимацию?**
A: Отредактируйте `generate_images_matplotlib.py`, добавьте функцию, и запустите её вновь.

**Q: Сайт не интерактивный?**
A: Убедитесь, что `script.js` находится рядом с `index.html`
