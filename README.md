# 🐢 Выставка Turtle Анимаций

Красивый веб-сайт для отображения анимаций, созданных с помощью Python Turtle Graphics.

## 📋 Описание

Этот сайт содержит галерею анимаций, которые были нарисованы с использованием модуля Python `turtle`. Каждая карточка показывает:
- 📸 Скриншот анимации
- 📝 Описание
- 👁️ Кнопку для просмотра кода (в модальном окне)

## 🎨 Анимации в галерее

1. **Спираль** - Геометрическая спираль с постоянно растущим размером
2. **Концентрические Круги** - Вложенные окружности с радужными цветами
3. **Звёзды** - 8 звёзд, расположенных по кругу
4. **Цветок** - Красивый цветок из вращающихся кругов
5. **Волна** - Плавная синусоидальная волна
6. **Многоугольники** - Вложенные правильные многоугольники
7. **Мандала** - Сложный симметричный узор
8. **Фрактальное Дерево** - Рекурсивное дерево с ветвями
9. **Снежинка** - Симметричная снежинка с кристаллическими ветвями

## 🚀 Быстрый старт

### Вариант 1: С готовыми изображениями (matplotlib)

```bash
# Установите зависимости
pip install matplotlib numpy

# Запустите скрипт для генерации изображений
python generate_images_matplotlib.py

# Откройте index.html в браузере
```

### Вариант 2: Оригинальные Turtle анимации (версия с turtle и PIL)

```bash
# Установите зависимости
pip install pillow

# Запустите скрипт
python generate_images.py

# Откройте index.html в браузере
```

## 💻 Технологии

- **Frontend**: HTML5, CSS3, JavaScript
- **Backend (для генерации)**: Python 3.x
  - `turtle` модуль (стандартный)
  - `matplotlib` (опционально)
  - `PIL/Pillow` (опционально)

## 🔧 Как использовать

1. **Просмотр сайта**: Просто откройте `index.html` в браузере
2. **Просмотр кода**: Нажмите кнопку "👁️ Посмотреть код" на любой карточке
3. **Копирование кода**: Нажмите "📋 Копировать код" в модальном окне
4. **Запуск анимаций**: Скопируйте код и запустите его в Python

## 🎯 Как добавить новую анимацию

1. Добавьте функцию в `generate_images_matplotlib.py` (или `generate_images.py`)
2. Добавьте код в `script.js` в объект `codeExamples`
3. Добавьте HTML карточку в `index.html`
4. Генерируйте изображение

## 📝 Пример использования кода анимации

```python
import turtle

screen = turtle.Screen()
screen.setup(width=800, height=800)
t = turtle.Turtle()
t.speed(0)

# Ваш код рисования здесь
# ...

screen.exitonclick()
```
